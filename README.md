# GraphicsSandbox

A repository that I use for playing around with real-time computer graphics.

# Acknowledgement
Some of this code was written while following TheCherno's OpenGL tutorial series. Some of the code is also inspired by a course I'm currently taking [NPGR019].
All source code that was inspired by code from others is credited in the source files.

# Libraries used
 - GLFW (https://www.glfw.org/)
 - GLAD (https://glad.dav1d.de/)
 - GLM (https://github.com/g-truc/glm)
 - STB, specifically stb_image.h and stb_image_write.h (https://github.com/nothings/stb)
 - TinyOBJLoader and TinyGLTF (https://github.com/tinyobjloader/tinyobjloader and https://github.com/syoyo/tinygltf)
 - nlohmann/json, as a dependency for TinyGLTF (https://github.com/nlohmann/json)

All the libraries are contained inside the third-party directory.
